# Ship Game

A 3D ship navigation game built with Three.js and Vite. Steer your ship through the ocean, collect crates, and navigate the open waters!

## Features

- Interactive ship steering with a wheel control
- Realistic water simulation
- 3D ship model with sails, mast, and deck
- Collect floating crates to score points
- Smooth camera movement following the ship
- Responsive design for desktop and mobile

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SnirRadomsky/ship-game.git
cd ship-game
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the local development URL (usually `http://localhost:5173`)

## How to Play

- **Desktop**: Click and drag your mouse to rotate the ship's wheel
- **Mobile**: Touch and drag to rotate the wheel
- Steer the ship to collect floating crates
- Each crate collected adds 10 points to your score

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Technologies Used

- [Three.js](https://threejs.org/) - 3D graphics library
- [Vite](https://vitejs.dev/) - Build tool and dev server
